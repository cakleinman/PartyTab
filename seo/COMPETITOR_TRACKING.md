## Splitwise Deep Dive

### Website Analysis
- **Title:** "Split expenses with friends. :: Splitwise"
- **H1:** Dynamic - cycles through "Less stress when sharing expenses with housemates/trips/partners/anyone"
- **Meta Description:** Emphasizes "free tool for friends and roommates"

### Navigation Structure
- Home
- Blog (content marketing)
- Press (social proof, backlinks)
- Calculators (SEO tools, backlink magnets)
- API (developer community)
- Jobs (authority signal)
- Pro (upsell)

### Content Strategy
- Developer blog with updates
- Use-case specific content
- Calculator tools (tip calculator, expense calculators)
- Press mentions/media kit

### SEO Metrics (Dec 2025)
| Metric | Value |
|--------|-------|
| Monthly visits | 2.93 million |
| Session duration | 4:43 |
| Pages per visit | 3.69 |
| Bounce rate | 28.2% |
| Referring domains | 6,260 |
| Backlinks | 260,920 |

### Weaknesses to Target
1. **Requires app download** - We don't
2. **Free tier has ads** - We're ad-free
3. **Daily expense limits on free** - We don't have limits
4. **Account required** - We allow instant tab creation
5. **Complex for simple use cases** - We're simpler

### Keywords They Rank For (to target)
- split expenses
- split bills with roommates
- expense tracker app
- how to split bills
- group expense calculator

## Monthly Competitor Check Template

### Questions to Answer Each Month:

1. **Did competitors add new pages?**
   - Check their sitemap
   - Note new blog posts
   - Look for new landing pages

2. **Did their rankings change?**
   - Check top-10 for main keywords
   - Note any drops we could capitalize on

3. **Did they get new backlinks?**
   - Monitor new mentions
   - Identify link-worthy content ideas

4. **Did they launch new features?**
   - Check their changelog/blog
   - Consider if we should match

5. **Did they run ads?**
   - Check Google Ads for keyword bidding
   - Note ad copy for messaging ideas
