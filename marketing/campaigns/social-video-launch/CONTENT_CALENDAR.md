# PartyTab Content Calendar & Strategy

## Content Pillars

### 1. Pain Point Content (40%)
Relatable moments about splitting expenses with friends
- "POV: You're calculating who owes what after the trip"
- "The group chat when someone finally asks for money back"
- "That one friend who 'forgot' their wallet"

### 2. Product Demo (30%)
Show how PartyTab solves the problem
- Quick walkthrough videos
- Before/after comparisons (spreadsheet chaos vs PartyTab simplicity)
- Feature highlights

### 3. Use Case Scenarios (20%)
Specific trip/event types
- Bachelor/bachelorette parties
- Ski trips / beach vacations
- Roommate expenses
- Group dinners

### 4. Behind the Scenes (10%)
Build connection with audience
- Building the app journey
- User feedback/testimonials
- Team updates

## Video Templates

### Template 1: POV Pain Point
```
HOOK (0-3 sec): "POV: [relatable situation]"
PROBLEM (3-8 sec): Show the chaos/frustration
SOLUTION (8-15 sec): PartyTab demo
CTA (15-18 sec): "Link in bio - it's free"
```

### Template 2: Comparison
```
HOOK (0-3 sec): "Stop using spreadsheets for group trips"
BEFORE (3-8 sec): Messy spreadsheet/Venmo chaos
AFTER (8-15 sec): Clean PartyTab interface
CTA (15-18 sec): "Try it free at partytab.app"
```

### Template 3: Quick Demo
```
HOOK (0-2 sec): "This app saved my friendship"
DEMO (2-12 sec): Create tab → Add expense → Show split
RESULT (12-15 sec): "Everyone knows exactly what they owe"
```
