# Features Showcase Campaign

Branded visual assets showcasing PartyTab's core features for Instagram, X/Twitter, and other social platforms.

## Assets

| File | Feature | Best For |
|------|---------|----------|
| `debt_simplification.png` | Before/after payment comparison | Hero image, explains core value |
| `live_balance.png` | Real-time net balance tracking | Shows app UI |

## Posting Tips

- **X/Twitter**: Post images directly, or convert to GIF for animation
- **Instagram Feed**: Use as-is (1:1 works best) or crop to 4:5
- **Instagram Stories/Reels**: Convert to MP4, 9:16 aspect ratio
- **TikTok**: Convert to MP4
