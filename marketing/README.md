   - @partytabapp on TikTok ✅ Available
   - @partytabapp on Instagram ✅ Available
   - @partytabapp on X/Twitter ✅ Available
   - u/partytabapp on Reddit ✅ Likely available

2. **Set up Product Hunt** "Coming Soon" page

3. **Record 2-3 TikTok videos** using scripts in `/content/TIKTOK_SCRIPTS.md`

### Week 2-4: Content & Community
4. Start posting content (see `/content/CONTENT_CALENDAR.md`)

## Free User Acquisition Channels (Priority Order)

### Tier 1: Highest ROI
1. **TikTok/Reels** - Viral potential, perfect for visual demos
2. **Product Hunt** - Launch moment, early adopter audience
3. **Reddit** - High-intent users asking about expense splitting

### Tier 2: Medium Effort, Good Returns
4. **SEO/Blog** - Long-term traffic (3-6 month payoff)
5. **Twitter/X** - Build in public, startup community
6. **Facebook Groups** - Targeted communities (bachelor parties, travel)

### Tier 3: Lower Effort, Supplementary
7. **Directory listings** - BetaList, AlternativeTo, etc.
8. **Hacker News** - Tech audience, good for feedback

## Content Quick Reference

### Voice & Tone
- **Casual & relatable** - Talk like a friend, not a brand
- **Slightly sarcastic** - Lean into the "money drama" humor
- **Helpful first** - Provide value before promoting

### Key Messages
1. "Split group expenses without the drama"
2. "No spreadsheets. No awkward Venmo requests."
3. "Free to use. No app download required."
4. "One payment settles everything."

### Visual Style
- Match website: green accent, cream/warm background
- Clean, minimal, friendly
- Show real app UI when possible
